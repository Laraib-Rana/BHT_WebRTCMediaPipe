# 🎛️ BHT WebRTCMediaPipe: Real-Time Stream Processing & Filter Suite

A high-fidelity real-time video stream manipulation platform built completely natively in the browser via WebRTC streaming interfaces and HTML5 canvas context maps. Created as an advanced module for the frontend development internship track.

## ✨ Features
* **Live WebRTC Ingestion Pipeline:** Secures, provisions, and tracks camera device permissions natively across modern browser clients.
* **Direct Pixel-Buffer Algorithms:** Computes frame arrays directly on the GPU to apply custom filters like Grayscale, High Inversion, and Neon Cyberpunk matrices instantly.
* **AR Cyberpunk Mask Overlays (Bonus):** Superimposes custom target vector graphic layers over real-time canvas coordinate points.
* **Integrated WebM Video Exporter (Bonus):** Uses the MediaRecorder API to record processed canvas operations and export them as a local download.

## 🛠️ Tech Stack
* HTML5 Native Video / Core Canvas Elements
* Responsive Bootstrap 5 Style Layouts
* WebRTC APIs (`getUserMedia` & `captureStream`)
* Vanilla JavaScript (Byte Buffer Traversals & Vector Matrix Calculations)
